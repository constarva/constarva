# Constarva - Technology Venture Studio

Managing, building, and scaling innovative technology ventures across AI, IoT, software, automation, and emerging tech.

## Technologies

- Vite
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn/ui

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
